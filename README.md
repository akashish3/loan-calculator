Monthly Payment is calculated using the standard loan amortization formula:

\[
M = \frac{P \cdot r \cdot (1+r)^n}{(1+r)^n - 1}
\]

Where:
- \(M\) = Monthly Payment  
- \(P\) = Loan Amount  
- \(r\) = Monthly Interest Rate (Annual Rate Ã· 12 Ã· 100)  
- \(n\) = Number of Months
